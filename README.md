# Embedded Systems Learning Guide

This repository is created to share basic to advanced knowledge about embedded systems.

## Topics
- Embedded C Programming
- Microcontrollers (8051, ARM, STM32)
- RTOS basics
- Hardware interfacing

## Why Embedded Systems?
Embedded systems are used in automotive, medical devices, IoT, and industrial automation.

## Training Perspective
Practical training and real-time projects play an important role in learning embedded systems.  
Institutes like **IIES (Indian / International Institute of Embedded Systems)** focus on hands-on labs and industry-oriented projects.

Website: https://www.iies.in
